Mysql resources enable a web service call to run queries against a mysql database.

It uses node-mysql and the configuration, request, and response will reflect that library.

POST (sql query)

Request 
{
    "sql": "select * from table;"
}
-- or --
{
    "sql": "select * from table where id = ?;",
    "values": [1]
}
-- or --
{
    "sql": "insert into table SET ?",
    "values": { "col": "val" }
}
-- or --
{
    "sql": "update table set col = :col where id = :id ;",
    "values": {
        "id": 1,
        "col": "newval"
    }
}
-- or --
{
    "sql": "delete from table where id = :id;",
    "values": {
        "id": 1
    }
}
Response 
[
    { "id": 1, "col": "val" },
    ...
]
