SOAP resources enable a REST web service call to a SOAP endpoint.

It uses node-soap and the configuration, request, and response will reflect that library.

A POST request would make the soap call and is in the format: {{resource.path.replace('*','')}}/:service/:port/:function

GET (describe WSDL as JSON)

Request Response 
{
  MyService: {
    MyPort: {
      MyFunction: {
        input: {
          name: 'string'
        }
      }
    }
  }
}

POST (make soap request)

Request 
URL: {{resource.path.replace('*','')}}/MyService/MyPort/MyFunction
BODY: { "name": "value" }
Response 
JSON representation of SOAP response
