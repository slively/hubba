PostgresSQL resources enable a web service call to run queries against a PostgresSQL database.

It uses node-postgres and the configuration, request, and response will reflect that library.

POST (sql query)

Request 
{
    "sql": "select * from table;"
}
-- or --
{
    "sql": "select * from table where id = $1;",
    "values": [1]
}
-- or --
{
    "sql": "insert into table SET col = $1;",
    "values": ["val"]
}
-- or --
{
    "sql": "update table set col = $2 where id = $1;",
    "values": [1, "newval"]
}
-- or --
{
    "sql": "delete from table where id = $1;",
    "values": [1]
}
