Email resources enable a web service call to send emails using the credentials entered into the configuration.

It uses nodemailer and the configuration, request, and response will reflect that library.

POST (send email)

Request 
{
    from: "me@domain.com",
    to: "you@otherdomain.com",
    subject: "Hello you!",
    text: "Plaintext body",
    html: "<h1>Big HTML body!</h1>"
}
Response 
success/error message
